### Что такое JSX
- JSX (JavaScript XML) — это синтаксис, который позволяет писать **HTML-похожие конструкции внутри JS-кода**.  
- По сути — это **функция `React.createElement()`**, но в более понятной и читаемой форме.  
- JSX используется в React, чтобы описывать, **как должен выглядеть интерфейс**.



## ⚙️ Правила хуков (Hooks Rules)

React хуки:
    useState()
    useEffect()
    useRef()
    useMemo()
    useCallback()
    useContext()

1. Хуки можно вызывать **только внутри React-компонента** или **кастомного хука**.  
2. Хуки нельзя вызывать:
   - ❌ внутри циклов (`for`, `while`)
   - ❌ внутри условий (`if`, `else`)
   - ❌ внутри вложенных функций
3. Всегда вызывать хуки **на верхнем уровне** функции-компонента.
4. React определяет состояние по **порядку вызова хуков**, а не по их названию.

## ⚡ Горячие клавиши в VS Code

**Редактирование:**
- `Ctrl + D` — выделить следующее совпадение (очень удобно для замены всех одинаковых слов)
- `Ctrl + /` — закомментировать или раскомментировать выделенный код
- `Shift + Alt + ↓` — дублировать строку вниз
- `Alt + ↑ / ↓` — переместить строку вверх или вниз
- `Ctrl + X` — вырезать всю строку (если ничего не выделено)
- `Ctrl + Shift + K` — удалить строку

**Навигация:**
- `Ctrl + P` — быстрый поиск и открытие файлов
- `Ctrl + Shift + O` — переход к функции / компоненту в файле
- `Ctrl + B` — скрыть / показать боковую панель
- `Ctrl + ~` — открыть или закрыть терминал

**Красота и порядок:**
- `Alt + Shift + F` — ручное форматирование (если вдруг не сработал Prettier)
- `Ctrl + S` — сохранить и автоформатировать (если включён Prettier)
